GOOMBAServer documentation

GOOMBAServer is a programming language for websites.

GOOMBAServer is SERVER SIDE! Like asp, PHP, and stuff. If you want to use GOOMBAServer you can use it.
Even though this language might not look as easy than other languages, it is still quite easy to learn.

This documentation may be inconsistent, but methods and functions are meant to be understood as the same.

Use GOOMBAServer

To use GOOMBAServer, you need to download the server. There is a Windows version and a Linux (Ubuntu) version
After you've done that, you can start creating your first index.goomba file in the /www folder.
<head>
 <title>
My first GOOMBAServer project!
</title>
 </head>

After doing that, you can add a server script anywhere on your site. To initiate GOOMBAServer, type:
->GOOMBATIME<-

To execute script, do this:
->GOOMBATIME<-
 Script here!
 ->STOPITGOOMBA<-

You can also add HTML code, just don't do it inside the GOOMBA tags. ( ->GOOMBATIME<- is the start of a GOOMBA script, ->STOPITGOOMBA<- is the end.)

Comments

GOOMBAServer ofcourse supports comments. To create a comment, start the line with THINK and it will be ignored. You cannot create multi-line comments.


Core reference

  • SPELLWORD: [word]: Writes the specified word on the page.
  • SPELLSENTENCE: [string]: Writes the specified string on the page.
  • WRITEPAPER: [filename] [string]: Opens/creates the file, and writes the specified string to the file.
  • APPENDTOPAPER: [filename] [string]: Opens/creates the file, and appends the specified string to the file.
  • READPAPER: [filename]: Opens the file and reads the files content and displays it on the screen, unless used in another way.
  • USEADVANCEDTEXT: : This is an extension mainly for READPAPER, adding this before the filename will change \n to a new line.
  • CONNECT: [host] [database name] [username] [password]: Connect to a MySQL database.
  • QUERY: "[query]" : Do something with the database.
  • CREATEHTMLVAR: [variablename] [elementid]: Reads the element by id, then stores it into a variable, similar to JS's document.getElementById
  • SPELLVAR: [variablename]: Write the variable on the page.
